How to run:
1. Import the Schema and Seed File (Located inside "DB" folder).
2. Open the project in IDE (The project is inside the "wasteland-survivors-backend.zip" file.
3. Adjust the "application.properties" file according to you database settings (Located in src->resources)
4. Run the "WastelandSurvivorsBackendApplication.java" file
5. Open the Unity project folder inside Unity
6. Navigate to Assets->Scenes
7. Run the "Login_register_page" file
8. Register a new Account. The seed Credentials are- Username: demo_guest, demo_owner; Password: password123 
9. Enjoy
